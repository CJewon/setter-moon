# SellerRoom

SellerRoom은 초기 1인 셀러가 상품, 옵션별 재고, 주문, 배송 상태를 한곳에서 정리하는 운영 관리 서비스입니다.

스마트스토어, 쿠팡, 인스타그램, 카카오톡, 오프라인 판매처럼 이미 쓰고 있는 판매 채널의 운영 정보를 모아 보여주는 내부 관리 도구입니다. SellerRoom은 판매 채널 자체를 만들거나, 결제와 정산을 대신 처리하는 서비스가 아닙니다.

## 누구를 위한 서비스인가요?

- 혼자 상품과 주문을 관리하는 1인 셀러
- 같은 상품을 색상, 사이즈, 용량, 구성 옵션으로 나눠 판매하는 셀러
- 주문은 여러 채널에서 들어오지만 재고는 따로 정리해야 하는 셀러
- 엑셀, 메모장, 쇼핑몰 관리자 화면을 오가며 주문 상태를 확인하는 셀러
- 복잡한 ERP보다 가볍게 시작할 수 있는 운영 도구가 필요한 초기 사업자

## 어떤 문제를 해결하나요?

- 옵션별 실제 재고를 한눈에 보기 어렵습니다.
- 주문접수 후 언제 재고를 차감해야 하는지 헷갈립니다.
- 오늘 처리해야 할 배송대기 주문을 놓치기 쉽습니다.
- 취소, 보류, 배송중 같은 주문 상태를 따로 기록하기 번거롭습니다.
- 여러 판매 채널의 운영 정보를 한곳에서 보기 어렵습니다.

## 주요 기능

- 상품 등록과 판매 상태 관리
- 색상, 사이즈 같은 옵션 조합별 재고 관리
- 주문 등록과 주문 상태 관리
- 주문접수 예약 수량과 가용 재고 확인
- 배송대기 전환 시 실제 재고 차감
- 재고 부족 상품 확인
- 주문과 재고 흐름을 보는 대시보드
- 계정 정보, 스토어 정보, 사용량 한도 관리

## 운영 방식

SellerRoom의 주문 흐름은 단순합니다.

```txt
주문접수
  -> 아직 실제 재고를 차감하지 않습니다.
  -> 주문 수량은 예약 수량으로 계산합니다.

배송대기
  -> 실제 재고를 차감합니다.

배송중 / 배송완료
  -> 배송 진행 상태를 확인합니다.

취소 / 보류
  -> 주문 상태를 정리합니다.
  -> 실제 환불, 교환, 반품 처리는 판매자가 사용하는 외부 채널에서 진행합니다.
```

## 무료 플랜

초기 1인 셀러가 부담 없이 시작할 수 있도록 무료 플랜을 기준으로 설계합니다.

```txt
상품: 10개
옵션 조합: 100개
월 신규 주문 등록: 300건
스토어 관리: 제공
재고 관리: 제공
배송 상태 관리: 제공
대시보드: 제공
```

유료 풀버전 후보 가격은 월 9,900원이며, 기능 차이는 최소화하고 사용량 한도를 해제하는 방향으로 기획하고 있습니다.

## MVP 범위

현재 MVP는 초기 셀러가 직접 입력하고 확인하는 운영 흐름에 집중합니다.

포함하는 범위:

- 랜딩, 회원가입, 로그인, 온보딩
- 대시보드, 마이페이지, 설정
- 상품 목록, 등록, 상세, 수정, 숨김
- 옵션 조합과 옵션별 재고
- 재고 목록, 부족 재고, 재고 이력
- 주문 목록, 등록, 상세, 수정
- 주문 단건/다건 상태 변경
- 무료 플랜 사용량 확인

제외하는 범위:

- 외부 판매 채널 API 자동 연동
- 결제 자동화
- 정산, 마진, 손익 계산
- 팀원 초대와 계정 공유
- 여러 스토어 전환
- 실제 택배사 송장 연동
- 교환, 반품, 환불의 금전 처리 자동화
